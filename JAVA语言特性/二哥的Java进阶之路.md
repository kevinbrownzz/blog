变量可以分为局部变量、成员变量、静态变量。

当变量是局部变量的时候，必须得先初始化，否则编译器不允许你使用它。

int 作为成员变量时或者静态变量时的默认值是 0。那不同的基本数据类型，是有不同的默认值和大

小的，来个表格感受下

| **数据类型** | **默认值** | **大小** |
| ------------ | ---------- | -------- |
| boolean      | false      | 1 比特   |
| char         | '\u0000'   | 2 字节   |
| byte         | 0          | 1 字节   |
| short        | 0          | 2 字节   |
| int          | 0          | 4        |
| long         | 0L         | 8        |
| float        | 0.0f       | 4        |
| double       | 0.0        | 8        |
|              |            |          |
